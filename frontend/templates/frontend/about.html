System Architecture

ScaleRec is a Tier-1, decoupled machine learning application demonstrating modern production pipelines. It separates the user-facing web layer from the GPU-intensive inference layer.

1. The Recommender Core (SASRec + DCN-V2)

The backend is powered by a two-stage deep learning pipeline built with TensorFlow Recommenders. It utilizes a Self-Attentive Sequential Recommendation (SASRec) model to capture temporal user behavior, passing latent vectors to a FAISS index for high-speed candidate retrieval. Candidates are then ranked using a Deep & Cross Network V2 (DCN-V2) to predict exact click-through/match rates.

2. Microservice Backend (FastAPI + Uvicorn)

The machine learning models are hosted on a lightning-fast, asynchronous Python backend via FastAPI. It exposes a strict REST API utilizing Pydantic data validation, ensuring the frontend only passes clean, tensor-ready arrays to the inference engine.

3. Decoupled Web Layer (Django + In-Memory Caching)

The frontend is an enterprise-grade Django application. It handles session security, CSRF protection, and user authentication. To achieve zero-latency UI rendering, the 60,000+ row dataset is loaded directly into a LocMemCache layer upon server initialization, bypassing disk I/O bottlenecks entirely.

4. LLM Agent Copilot (Gemini 2.5 Flash)

To overcome the temporal limitations of the static 25M movie dataset, a generative AI agent is integrated into the dashboard. It acts as an entertainment copilot, capable of retrieving information on modern releases, aggregating reviews, and generating actionable booking links dynamically.
