## Docker Quick Start Guide
1. Install Docker on your machine.
2. Clone the repository: `git clone https://github.com/Arjun-Patel1/recsys2.0`
3. Navigate to the project directory.
4. Build the Docker image: `docker build -t recsys .`
5. Run the Docker container: `docker run -p 5000:5000 recsys`

## Local Development Setup Instructions
1. Clone the repository: `git clone https://github.com/Arjun-Patel1/recsys2.0`
2. Navigate to the project directory.
3. Create a Python virtual environment: `python -m venv venv`
4. Activate the virtual environment: `source venv/bin/activate`
5. Install dependencies: `pip install -r requirements.txt`
6. Run the application: `python app.py`
